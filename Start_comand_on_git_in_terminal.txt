I. Команды для локальной работы на компьютере.

	1. cd (Команда, позволяет пекремещаться между директориями)
	2. pwd (показывает директорию, в которой мы находимся)
	3. ls (Показывает содержимое директории)


II. Команды Git.

	1. git status (Данная команда показывает нам статус нашей локальной директории)
	2. git add <file_1> <file_2> ... (Добавляем файлы для подготовки коммита)
	3. git commit (Подготовка коммита к загрузке (Создание сообщения коммита))
	4. git push (Загрузка коммита в репозиторий)
